# 🚀 DigiTools

## 📌 About The Project

**DigiTools** is a modern web application that offers a collection of powerful AI tools to enhance productivity and simplify digital tasks. Users can easily explore and use different AI tools from a single, user-friendly platform with a smooth and responsive experience.

---

## ⚡ Technologies Used

* ⚛️ React.js
* 🟨 JavaScript (ES6+)
* 🎨 Tailwind CSS
* 📦 JSON (Static Data)
* 🎯 React Icons
* ✨ Lucide React Icons
* 🔔 React Toastify

---

## ✨ Features

* 🔍 **Explore AI Tools**
  Browse and discover various AI tools with a clean and intuitive interface.
* ⚡ **Responsive Design**
  Fully responsive across mobile, tablet, and desktop for a seamless user experience.
* 🔔 **Toast Notifications**
  Real-time feedback and alerts using React Toastify for better interaction.

---

## 🚀 Getting Started

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/digitools.git
```

### 2️⃣ Go to project directory

```bash
cd digitools
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```
